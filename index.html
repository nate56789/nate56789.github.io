Definitions

Computational Social Choice is the field combining many different areas of study. These areas include theoretical computer science, social choice theory, and more.

Theoretical Computer Science (in very brief summary) is a branch of computer science that focuses more on the mathematical components of computer science rather than just applications of these components.

Social Choice Theory is a type of methodology for making equitable decision-making. Specifically, these decisions are based on user preferences, opinions, and interests and potentially other characteristics. This decision that is reached needs to be more or less collective.

Hopefully those definitions gave you some idea of where we are going in this chapter. Computational social choice is seen all over the place and this chapters will show you some of those examples as well as the underlying mathematical notion of how those examples can be expressed. Mainly, this chapter will be concerned with making decisions in elections of various forms and analyzing the pros and cons of different election rules. Let us begin with the simplest definition which is that of a voter.
Definitions
A Voter is simply an entity who votes in an election.
This definition directly follows from the real-world concept of a voter. The next few definitions will similarly follow from real-world concepts.
Definitions
A Candidate/Alternative is an entity that can be voted for in an election.

A Preference Ballot/Profile/List is a ranking of the candidates.

Each voter constructs a preference list.
Now that we established how an election can be described, we can discuss different rules in these elections and the pros and cons of each type of rule.
Definitions (Voting Rules)
The Plurality Rule for voting is when we only consider the top-ranked candidate for each voter. Then the candidate that is top-ranked the most times is the winner of the election.
Real Life Example
The plurality rule is essentially what the United States uses for its elections. Each state lists its first preference choice and then across all states the candidate with the most first choice across each state (multiplied by some electoral college weight for population) is the victor. Note that this does not necessarily imply that the winning candidate got the majority of the votes/is the most preferred.
Example
Consider the following table of voting where the first row indicates the number of people with that permutation preference list for those candidates.
2 4 4 4
Alice Alice Bob Charlie
Bob Charlie Charlie Alice
Charlie Bob Alice Bob
Then Alice gets 6 first choice votes, Bob gets 4, and Charlie gets 4. So, Alice would be the winner. However, note that Alice would not win a head to head with both Bob and Charlie even though she wins against both at the same time. Consider Alice vs Charlie. Then Alice would get $(2+4) = 6$ first place votes but Charlie would get $(4+4) = 8$ first place votes. So, Charlie would have won if Bob was not present. We will explore what this property is called below.
Definitions (Voting Rules)
The Instant Runoff/Plurality With Elimination for voting is when each round, we eliminate the candidate with the least first place votes. This continues until a majority is reached for a specific candidate. That candidate is declared the winner of the election.
Real Life Example
This voting rule is actually used for the Australian House of Representatives election.
Example
Using a similar table to previous example
1 2 4 6
Alice Alice Bob Charlie
Bob Charlie Charlie Alice
Charlie Bob Alice Bob
Note that there are a total of 13 voters so it takes 7 votes to reach a majority. In the first round Charlie gets 6 first place votes so a majority is not reached. Therefore we eliminate the person with least first-place votes which in this case would be Alice. Then in the next round with just Charlie and Bob, Charlie gets $2+4+6 = 12$ first-place votes and Bob would get just 1 first-place vote. Therefore Charlie would be declared the winner since he got the majority.
Definitions (Voting Rules)
The similar Plurality With Runoff Rule for voting is when you take the first and second place winner based on the number of first-place votes and pit them against each other.
Example
For the previous table Alice has $3$ first place votes, Bob has $4$ and Charlie has $6$. So since Bob and Charlie are in second and first place respectively, they would go against each other and the end result would be the same as in the previous example.
Definitions (Voting Rules)
The Borda Count voting rule is another voting rule where for each voter's preference list, 1 point is given to the last place candidate, 2 to second to last, 3 to third to last, and so on. Then the candidate with the most amount of points at the end is declared the winner.
Example
Consider the following table of voting where the first row indicates the number of people with that permutation preference list for those candidates.
2 4 4 5
Alice Alice Bob Charlie
Bob Charlie Charlie Alice
Charlie Bob Alice Bob
In this table there are 3 candidates, so first-place gets 3 per voter, second-place 2, and third-place 1. Then for this table Alice gets $2*3 + 4*3 + 1*4 + 2*5 = 6+12+4+8=32$ for column 1,2,3,4 respectively. Using the same method Bob gets $2*2 + 4*1 + 4*3 + 5*1 = 25$ and finally Charlie gets $2*1 + 4*2 + 4*2 + 5*3 = 33$. So Charlie would be declared the winner. Note that if we were using the plurality rule then Alice would be declared the winner.
Real Life Example
The Borda Count is used all over the place. One instance is the selection of the MVP in Major League Baseball.
